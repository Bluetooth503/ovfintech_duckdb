# Gompertz NLME 模型收敛问题分析报告

## 一、模型配置

| 配置项 | 值 |
|--------|-----|
| 模型类型 | PyMC 贝叶斯 NLME Gompertz |
| 建模牛只 | 731 头（分层采样） |
| 固定效应 | 品种(1) + 牧场(8) + 饲料(4) = 14 个（含截距） |
| 随机效应 | 个体级 A/b/C 三参数偏差 |
| 随机效应总维度 | 731 × 3 = 2,193 |
| MCMC 配置 | 8链, 1000采样, 500预热, target_accept=0.95 |
| 先验参数 | A~log(650)kg, b~log(0.015)/day, C~225天 |

## 二、模型诊断：未收敛

| 诊断指标 | 实际值 | 达标标准 | 评估 |
|---------|--------|---------|------|
| ESS_min | 15 | >100 | 严重不足 |
| R-hat_max | 1.47 | <1.05 | 未收敛 |
| Divergences | 3,284 | 0 或极少 | 严重发散 |
| 平均接受率 | 0.51 | 接近目标0.95 | 远低于目标 |
| 最大树深度 | 14 | 接近上限15 | 采样效率低 |

## 三、参数估计（不可靠）

| 参数 | 先验 | 估计值 | 95% CI |
|------|------|--------|--------|
| A（成熟体重） | 650 kg | **1,191 kg** | [916, 1,617] |
| b（生长速率） | 0.015/day | **0.00269** | [0.00156, 0.00499] |
| C（拐点日龄） | 225 天 | **167 天** | [128, 212] |

三个参数均显著偏离先验，且可信区间过宽，估计不可靠。

## 四、随机效应方差分解

| 随机效应 | 方差 | 标准差 | 含义 |
|---------|------|--------|------|
| u_log_A | 0.05 | 0.23 | 个体成熟体重偏差 ±26%，**合理** |
| u_log_B | 0.10 | 0.32 | 个体生长速率偏差 ±38%，**合理** |
| **u_C** | **15,541** | **124.7 天** | 个体拐点日龄偏差 ±125 天，**严重失控** |

A 和 b 的随机效应正常，**问题完全集中在 C（拐点日龄）的个体偏差**。

## 五、根因分析：称重数据无法支撑个体级拐点日龄估计

### 5.1 称重间隔极度稀疏

| 相邻称重间隔 | 次数 | 占比 |
|-------------|------|------|
| ≤7天 | 2,867 | 7.8% |
| 8-30天 | 3,777 | 10.3% |
| 31-60天 | 4,170 | 11.4% |
| 61-90天 | 2,958 | 8.1% |
| **>90天** | **22,792** | **62.3%** |

超过六成的相邻称重间隔在 90 天以上。平均每 3 个月才称一次重。

### 5.2 称重日龄分布不均匀

| 日龄区间 | 称重次数 | 占比 | 生长阶段 |
|---------|---------|------|---------|
| <100天 | 50,945 | **33.8%** | 幼龄期 |
| 100-200天 | 22,379 | 14.8% | 拐点前 |
| **200-300天** | **18,789** | **12.4%** | **拐点区间** |
| 300-400天 | 22,148 | 14.7% | 拐点后 |
| 400-500天 | 10,473 | 6.9% | 育肥期 |
| 500-600天 | 8,128 | 5.4% | 育肥后期 |
| >600天 | 18,058 | 12.0% | 成熟期 |

1/3 的称重集中在幼龄期（<100天），拐点区间（200-300天）仅占 12.4%，是所有区间中覆盖最少的。

### 5.3 拐点区间覆盖严重不足

在观测≥3次的 6,145 头牛只中：

| 拐点区间(100-300天)观测次数 | 牛只数 | 占比 |
|---------------------------|--------|------|
| **0次（完全无覆盖）** | **2,194** | **35.7%** |
| 1次 | 2,591 | 42.2% |
| ≥2次 | 1,360 | 22.1% |

77.9% 的牛只在拐点区间观测不足 2 次。**拐点前后均有观测的牛只仅 785 头，占 12.8%。**

### 5.4 各日龄区间的牛只覆盖率

| 日龄区间 | 有称重记录的牛只占比 |
|---------|-------------------|
| 0-100天 | 38.6% |
| 100-200天 | 19.1% |
| **200-300天** | **15.4%** |
| 300-400天 | 18.4% |
| 400-600天 | 15.3% |
| >600天 | 13.3% |

拐点区间（200-300天）仅 15.4% 的牛只有称重记录，是所有区间中覆盖率最低的。

## 六、因果链

```
称重间隔稀疏（62% >90天）
    ↓
拐点区间覆盖不足（77.9% 牛只≤1次观测，仅12.8%前后均有观测）
    ↓
个体级拐点日龄 C 不可识别（数据无法区分拐点是180天还是250天）
    ↓
u_C 方差失控（sd=125天，模型自由搜索巨大参数空间）
    ↓
后验几何病态（窄谷+高维，NUTS步长无法调优）
    ↓
Divergences=3284，接受率=0.51，R-hat=1.47
    ↓
模型不收敛，参数估计不可靠
```

## 七、为什么 MIN_OBS=4 反而更差

| 指标 | MIN_OBS=3 | MIN_OBS=4 |
|------|-----------|-----------|
| Divergences | 1,298 | **3,284**（+153%） |
| 接受率 | 0.79 | **0.51**（-35%） |
| ESS_min | 17 | **15** |

更多观测使个体参数被数据"拉离"群体均值，后验从平坦变为尖峰，NUTS 采样器在尖锐的后验分布中更难找到合适步长。但 u_C 方差仍然失控（15,075 vs 15,541），说明问题不在数据量，而在于数据结构（稀疏+不覆盖拐点）从根本上无法支撑个体级 C 估计。

## 八、结论

模型不收敛的根因是**称重采样频率不足以支撑个体级拐点日龄估计**。在当前称重频率（平均 3 个月一次）下，仅 12.8% 的牛只有足够数据锚定拐点位置。这不是模型设计的问题，而是数据结构与模型复杂度的不匹配。
