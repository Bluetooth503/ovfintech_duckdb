# 中国肉牛 Gompertz NLME 多维度生长曲线混合效应模型先验参数分析报告

## 1. 引言
本报告旨在为中国肉牛 Gompertz 非线性混合效应（NLME）生长曲线模型提供最佳先验超参数值。通过对互联网文献和数据的宽泛检索，我们重点分析了成熟体重（A）、生长速率（b/k）、拐点日龄（C）以及 LKJ 相关系数先验参数（LKJ_ETA），并探讨了品种和饲料结构等固定效应对模型参数的影响。

## 2. 先验参数分析与推荐

### 2.1 成熟体重先验参数 (PRIOR_A_MU, PRIOR_A_SIGMA)
**初始设定**：PRIOR_A_MU = np.log(532) kg，PRIOR_A_SIGMA = 1.5
**文献发现**：多项研究表明，中国肉用西门塔尔牛的成熟体重普遍高于初始设定的 532 kg。段星海等（2021）的研究显示，中国肉用西门塔尔牛公牛的平均成熟体重为 629.11 kg，标准差为 88.71 kg [1]。其他文献也指出，西门塔尔牛的成熟体重通常在 600-800 kg 之间。
**推荐值**：
- **PRIOR_A_MU = np.log(650)**：将均值调整至 650 kg，能更好地反映中国肉用西门塔尔牛的实际成熟体重中位数。
- **PRIOR_A_SIGMA = 1.0**：将对数尺度上的标准差调整为 1.0，既能覆盖大部分观测值，又避免了过宽的先验分布。

### 2.2 生长速率先验参数 (PRIOR_B_MU, PRIOR_B_SIGMA)
**初始设定**：PRIOR_B_MU = np.log(0.015) /day，PRIOR_B_SIGMA = 1.5
**文献发现**：国际上关于牛 Gompertz 模型生长速率参数（以每日速率计）通常在 0.005 至 0.03 之间。初始设定的 0.015 /day 恰好位于这一合理区间的中部。
**推荐值**：
- **PRIOR_B_MU = np.log(0.015)**：保持不变，具有较强生物学合理性。
- **PRIOR_B_SIGMA = 0.5**：初始的 1.5 过于宽泛，调整为 0.5 后，95% 置信区间约为 [0.0056, 0.029]，与国际文献中的典型取值范围高度吻合。

### 2.3 拐点日龄先验参数 (PRIOR_C_MU, PRIOR_C_SIGMA)
**初始设定**：PRIOR_C_MU = 200 天，PRIOR_C_SIGMA = 80 天
**文献发现**：Duan 等（2021）的研究计算得出拐点日龄约为 194.39 天 [1]。Liang 等（2018）的研究报告拐点日龄约为 254.1 天 [2]。
**推荐值**：
- **PRIOR_C_MU = 225**：取两项研究的平均值，更贴近实际数据。
- **PRIOR_C_SIGMA = 30**：考虑到研究间的差异，30 天的标准差能较好地反映变异性，且比初始的 80 天更精确。

### 2.4 LKJ 相关系数先验参数 (LKJ_ETA)
**初始设定**：LKJ_ETA = 2.0
**文献发现**：在贝叶斯建模中，弱信息先验通常是首选。$\eta = 1.0$ 的 LKJ 先验在相关矩阵空间上提供了一个均匀分布，避免了引入过强的先验假设 [3]。
**推荐值**：
- **LKJ_ETA = 1.0**：为了保持先验的弱信息性，并避免不必要的正则化，推荐选择 1.0。

## 3. 固定效应分析

### 3.1 品种固定效应
研究表明，不同品种的生长参数存在显著差异。以西门塔尔牛为例，其成熟体重均值为 629.11 kg，达到最大生长率的时间（b）均值为 2.82，成熟率（K）均值为 0.16 [1]。这些数据可作为 Gompertz NLME 模型中品种固定效应的参数化依据。

### 3.2 饲料结构特征固定效应
当前文献中缺乏直接量化饲料结构（精粗比、能量蛋白水平）和牧场管理模式对 Gompertz 模型参数固定效应的精确数值。建议采用弱信息先验或专家经验结合间接证据进行定性推断。

## 4. 结论
通过对文献的综合分析，我们对中国肉牛 Gompertz NLME 模型的先验超参数进行了优化推荐。这些推荐值基于实际观测数据和贝叶斯建模的最佳实践，能够为模型的准确拟合提供更可靠的先验信息。
