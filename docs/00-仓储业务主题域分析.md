#### 业务价值

#### 设备类型

- **盘料仪**: 固定式/手持式库存盘点设备，用于质押物库存核实
- **摄像头**: 监控安防设备，实时监控质押物状态
- **道闸**: 车辆进出控制设备，管控质押物流动
- **称重设备**: 地磅、台秤等计量设备，精确计量质押物重量
- **传感器**: 温湿度、烟感等环境监控设备

#### 供应链金融风控价值

| 风控场景 | 设备作用 | 风险指标 |
|----------|----------|----------|
| **质押物真实性** | 盘料仪自动盘点，验证库存数据 | 盘点差异率、设备在线率 |
| **质押物安全** | 摄像头24小时监控，异常自动告警 | 监控覆盖率、告警响应时间 |
| **质押物流动** | 道闸控制车辆进出，防止私自出货 |  unauthorized出库拦截率 |
| **环境风险** | 温湿度传感器监控存储条件 | 环境异常告警次数 |
| **设备可靠性** | 心跳监测设备在线状态 | 设备离线时长、故障恢复时间 |

#### 设备状态与风控等级

```
设备正常 → 监控有效 → 质押物安全 → 风险可控
设备离线 → 监控盲区 → 质押物风险 → 触发预警 → 冻结额度
```

- **设备在线率**: 直接影响质押融资的可用额度
- **盘点准确率**: 决定质押物估值的可信度
- **监控覆盖率**: 影响风险预警的及时性

```
AI监控 → 预警发现 → 工单派发 → 现场处理 → 结果反馈 → 巡检验证
```

## 🔍 业务架构特点

### 1. 金融仓储模式

- **质押监管**: 以库存商品为质押物的供应链金融服务
- **动态监管**: 支持库存动态变动的监管模式
- **风险预警**: 价格监控、库存异常、设备离线等多维度风控

### 2. 物联网深度融合（设备即风控基础设施）

- **设备联网**: 盘料仪、摄像头、道闸、地磅等设备统一接入
  - **质押物盘点**: 盘料仪自动盘点，确保质押物账实相符
  - **实时监控**: 摄像头24小时监控质押物安全状态
  - **流动管控**: 道闸控制车辆进出，防止私自转移质押物
- **AI智能**: 智能识别、自动盘点、异常预警
- **数据同步**: 实时库存同步、设备状态监控
- **设备风控联动**: 设备离线自动触发风控预警，冻结融资额度

### 3. 精细化仓储管理

- **库位级管理**: 精确到xyz坐标的货位定位
- **批次追踪**: 完整的批次追溯体系
- **条码管理**: 一品一码，全程可追溯

### 目录结构

按**数据仓库分层**规划目录，通过**模型命名**区分业务主题域。

```
models/
├── wms/                    # 仓储业务域
│   ├── ods/                # ODS层: 原始数据存储
│   ├── dim/                # DIM层: 维度表

│   ├── dws/                # DWS层: 汇总表

├── common/                 # 公共维度表(跨业务域共享)
│   └── dim/
└── docs/                   # 单据表单域(如有需要)
```

### 命名规范

| 层级 | 命名格式 | 示例 |
|------|----------|------|
| ODS | `ods_{源表名}` | `ods_warehouse`, `ods_inventory` |
| DIM | `dim_{业务域}_{实体}` | `dim_wms_warehouse`, `dim_wms_position`, `dim_wms_sku` |

**类型标识**: `trx`-事务明细, `snap`-快照, `agg`-聚合, `state`-状态

## 📝 总结

仓储业务是一个典型的**物联网 + 供应链金融 + 精细化运营**的复合型业务系统：

1. **资产数字化**: 从入库到出库的全流程条码追踪
2. **风控智能化**: AI监控 + 价格监控 + 库存监控 + 设备监控的多维风控体系
3. **金融场景化**: 基于库存商品的供应链金融产品
4. **运营精细化**: 库位级管理、批次级追踪、实时库存同步
5. **设备即风控**: 仓库设备是供应链金融风控的重要基础设施

### 仓库设备与供应链金融风控

在金融仓储场景中，**设备即风控能力**：

| 风控层级 | 设备能力 | 业务价值 |
|----------|----------|----------|
| **第一层：盘点核实** | 盘料仪自动盘点 | 确保质押物账实相符，防止虚假质押 |
| **第二层：实时监控** | 摄像头全覆盖 | 24小时监控质押物状态，及时发现异常 |
| **第三层：流动管控** | 道闸联动控制 | 防止未经授权的质押物转移 |
| **第四层：环境保障** | 温湿度传感器 | 确保质押物存储环境符合要求 |
| **第五层：状态感知** | 设备心跳监测 | 设备离线自动触发风控预警 |

### 数据仓库建设重点

- **库存流动性**分析(入库、在库、出库周转)
- **金融风险**监控(质押率、库存异常、价格风险、设备风险)
- **设备风控**能力(设备在线率、盘点准确率、监控覆盖率)
- **运营效率**提升(出入库效率、盘点准确率、设备利用率)
- **园区协同**优化(车辆周转、排队效率、道闸通行)
